# SmartTube Builder

Public builder for [yuliskov/SmartTube](https://github.com/yuliskov/SmartTube) — ad-free YouTube client for Android TV.

## Usage

1. Go to **Actions** → **Build SmartTube** → **Run workflow**
2. Wait a few minutes for the build to finish
3. Download the APK from the workflow run artifacts

## Features

- No ads
- SponsorBlock integration
- 8K / 60fps / HDR support
- Adjustable playback speed
- View live chat
- No Google Services required
